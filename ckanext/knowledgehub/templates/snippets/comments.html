Used to render the user name and gravatar. * 'enable_multilevel_replies', optional, if set to True, the replies can go multiple levels - there can be a reply to a reply and so on. This is rendered appropriately. If set to False, then all replies are to the top level comment and a Reply to a reply just adds the comment as a last reply to the top level comment. #} {% resource 'knowledgehub/comments' %} {% set enable_multilevel_replies = enable_multilevel_replies if not none else False %}
